## Описание

Проект содержит несколько рабочих страниц:

- Главная
- О нас
- Профиль

Доступ к профилю есть только у авторизованных пользователей. В профиле содержится информация об актуальных записях пользователя и форма для оформления услуги.

**В планах:** расширить функционал сайта, добавить защищённую авторизацию и каталог товаров для клиентов.

## Технологии

- **Языки:** Python 3.13, JavaScript
- **Фреймворки:** React, Flask
- **Инструменты:** Git, Docker
- **База данных:** PostgreSQL

## Установка и запуск

### Требования
- Docker
- Git

### Клонирование репозитория

```bash
git clone git@github.com:amainn/vetsite_pr.git
cd vetclinic-pr
```
## Доступ к сервисам
```bash
docker-compose up --build
```

После запуска контейнеров, сайт будет находиться по адресу: http://localhost

## Остановка
```bash
docker-compose down
```
